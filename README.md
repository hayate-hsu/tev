# TEV 
基于给定文本，搜索给定目录下的图片、视频文件，找到相近的图片或视频片段。

## install
建议python> 3.8 版本。
- 安装torch,详情参考[pytorch](https://pytorch.org/)官方
```
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

- 安装依赖库

```shell
pip install -r requirements.txt
```
